# Chart Selection Justification

## 1. Sales Trend

**Business Question:**  
How are sales changing over time?

**Chart Used:**  
Line Chart

**Reason for Selection:**  
A line chart is the most effective way to display trends and patterns over time, allowing leadership to identify seasonal changes and sales fluctuations.

**Fields Used:**
- X-Axis: Order Date (Month)
- Y-Axis: Sales

**Design Principle Applied:**
- Clear chronological ordering
- Minimal clutter
- Easy trend identification

**Mistake Avoided:**
Did not use pie charts or bar charts for time-series data.

---

## 2. Regional Performance

**Business Question:**  
Which region performs best?

**Chart Used:**  
Horizontal Bar Chart

**Reason for Selection:**  
Bar charts allow easy comparison of sales across different regions.

**Fields Used:**
- Category: Region
- Measure: Sales
- Color: Profit

**Design Principle Applied:**
- Sorted bars
- Consistent color usage
- Easy comparison

**Mistake Avoided:**
Avoided using complex charts that make regional comparison difficult.

---

## 3. Category Profitability

**Business Question:**  
Which product categories generate the highest profit?

**Chart Used:**  
Horizontal Bar Chart

**Reason for Selection:**  
Bar charts clearly compare profitability across categories and sub-categories.

**Fields Used:**
- Category: Sub-Category
- Measure: Profit
- Color: Category

**Design Principle Applied:**
- Color grouping
- Descending order
- Clear labels

**Mistake Avoided:**
Avoided overcrowded visualizations.

## 5. Shipping Performance

**Business Question:**  
Which shipping mode has the longest delivery time?

**Reason for Selection:**  
Makes comparison of average delivery days straightforward.

**Fields Used:**
- Category: Ship Mode

---

## 6. Discount vs Profit

**Business Question:**  
How does discount affect profit?

**Chart Used:**  
Scatter Plot

**Reason for Selection:**  
Scatter plots are ideal for identifying relationships between two numerical variables.

**Fields Used:**
- X-Axis: Discount
- Y-Axis: Profit
- Color: Category
- Size: Sales

**Design Principle Applied:**
- Shows distribution
- Highlights relationships
- Supports outlier detection

**Mistake Avoided:**
Avoided line charts, which are unsuitable for showing relationships between independent observations.
